## modules

- __Type_-: `Array<PolygenModuleCommonConfig>`

### Local modules

```ts title="polygen.config.mjs"
import {
  localModule,
  polygenConfig,
} from '@callstack/polygen-config';

export default polygenConfig({
  modules: [
    localModule('path/to/my-module.wasm')
  ],
});
```
