## modules

- __Type__: `PolygenModuleCommonConfig[]`
- __Default__: `[]`

List of modules to be generated.

```ts title="polygen.config.mjs"
import {
  localModule,
  polygenConfig,
} from '@callstack/polygen-config';

export default polygenConfig({
  modules: [
    localModule('path/to/my-module.wasm') // [!code highlight]
  ],
});
```
